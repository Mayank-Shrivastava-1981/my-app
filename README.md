# Test Automation Generator

This is a full-stack application with a Python backend and a React frontend.

## Project Structure

- `backend/`: FastAPI backend service (Python)
- `frontend/`: React frontend application (JavaScript)

## Getting Started

### Backend

1. Navigate to `backend/`
2. Create virtual environment: `python -m venv .venv`
3. Install requirements: `pip install -r requirements.txt`
4. Run server: `uvicorn server:app --reload`

### Frontend

1. Navigate to `frontend/`
2. Install dependencies: `npm install`
3. Run development server: `npm start`
